How did the 'process over product' concept affect the way you tackled the site redesign and rebuild?

The rebuild was a challenge for me, it took me awhile to familiarise myself with the skeleton framework. I discovered that my initial wireframe drawings were somewhat ambitious. I struggled to implement these changes without using javascript. I found a way to achieve the design I wanted using javascript, the problem was, I don't yet understand javascript. I had to remind myself about "process over product", and that there was no point in including code that I don't understand in my website. Definitely a learning curve! .

What did you think about meditation before reading chapter 2?

Utilising meditation is quickly becoming a strategy for me. I find that when I get stuck with my code, the best thing to do is take a break. This is when I often practice mindfulness, it's always a challenge escaping all the code problems that are running through my mind. But stepping back, and seeing my thoughts from a distance, watching them pass through my mind like balloons, definitely, helps me to relax.
