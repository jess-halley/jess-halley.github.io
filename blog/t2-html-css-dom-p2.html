What is grid based design?

Grid based design a methodology utilises a grid structure to break up, section and organise web page content. This design is effective because all elements on a page are boxed.

What is all the hype about responsive web pages?

Responsive web pages are a crucial to well-designed websites. The various interfaces we utilise to access the internet continue to change. Screen resolutions are increasing and more people access the internet is now more frequently accessed through mobile phone technology. This means that screen sizes can vary dramatically and consequently, that the standard page design of 1024 x 768 monitor is no longer appropriate. Practising responsive design is a way of ensuring that sites can dynamically adjust according to screen size or device type.

What is semantic structure?

Semantic structure is a way of organising HTML code so that it is easily read and followed by humans as well as computers. For instance, IMG is a semantic tag is easily recognised by English speakers to mean image, yet it is also interpreted by the browser.
